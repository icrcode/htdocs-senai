Sobre o Curso

🎯 O que você vai aprender

  • Lógica de programação e algoritmos
  • Desenvolvimento Web com HTML, CSS e JavaScript
  • Back-end com PHP e banco de dados SQL
  • Programação orientada a objetos
  • Versionamento de código com Git e GitHub
  • Desenvolvimento de sistemas e aplicações
  • Metodologias ágeis — Scrum e Kanban

🛠️ Tecnologias do Ambiente Local

  • Apache HTTP Server
  • PHP 8.0 — linguagem back-end principal
  • MariaDB — banco de dados relacional
  • phpMyAdmin — administração do banco
  • XAMPP Control Panel — gerenciador de serviços
  • Git — controle de versão
  • VS Code — editor de código recomendado
